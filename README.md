# Driver App

Expo React Native app for drivers to manage journeys and broadcast location.

## Setup

```bash
npm install
npx expo start
```

## Environment
- Copy .env.example to .env and fill required keys.

## Tests
- Unit tests: `npm test`
- Planned: `test:unit`, `test:e2e`
